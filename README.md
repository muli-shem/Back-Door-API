# G-NET Backend API

Generational Entrepreneurs Network - Complete Django REST Framework Backend

## Features

- JWT Authentication with refresh tokens
- Role-based access control (Member, Executive, Admin)
- Members management and profiles
- Finance/MMF system for member contributions
- Blessed Mind ideas and project proposals
- Organization announcements and events
- Membership applications management
- Comprehensive API documentation (Swagger UI)
